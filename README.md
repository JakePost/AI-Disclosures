# AI-Disclosures
A useful collection of text statements that can be used to disclose an author's use (or non-use) of generative AI.

## No AI Affirmation

> The authors affirm that generative artificial intelligence (AI) was not used in any stage of writing and production of this manuscript. 

> The authors affirm that generative artificial intelligence (AI) was not used to produce the code contributions detailed by this manuscript. 

## No Training Request

> We respectively request that the content of this manuscript is not used to train generative models. 

## AI Use Disclosure

> The authors wish to disclose that generative artificial intelligence (AI) was used throughout the production of this manuscript.


### No generative AI was used in the writing or editing of this file.
